# Portafolio Angular

Proyecto Angular que inicia la migración del portafolio previamente construido en React. Incluye componentes standalone, ruteo configurado en `main.ts`, formulario de contacto conectado al endpoint Express `/contact` y contenido base para portafolio.

## Scripts
- `npm start`: levanta el servidor de desarrollo.
- `npm run build`: genera la compilación de producción en `dist/`.
- `npm test`: ejecuta las pruebas configuradas por Angular CLI.
